# Calculadora React

Calculadora con estética Tron construida con React, TypeScript y Vite.

## Instalación

```bash
bun install
```

## Correr la aplicación

```bash
bun run dev
```

## Correr los tests

```bash
bun run test
```

## Correr Storybook

```bash
bun run storybook
```

## Lint

```bash
bun run lint
```
